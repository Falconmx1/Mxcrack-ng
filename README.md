# 🔓 Mxcrack-ng

> Herramienta de seguridad WiFi estilo aircrack-ng pero sin necesidad de modo monitor.

[![Python](https://img.shields.io/badge/Python-3.6%2B-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 🚀 Características

- 🔍 Captura de paquetes en redes cercanas
- 📡 Prueba de seguridad en redes WiFi
- ⚔️ Ataques de desautenticación (simulados)
- 🛠️ Herramientas adicionales (escáner, ping, reportes)
- ⚙️ Configuración de interfaz y tiempo
- 🎨 Menú interactivo con diseño mejorado
- ❓ Ayuda integrada

## 📋 Menú

1. 🔍 Capturar paquetes
2. 📡 Probar redes
3. ⚔️ Ataques
4. 🛠️ Herramientas
5. ⚙️ Opciones
6. ❓ Ayuda
7. 🚪 Salir

## 🛠️ Requisitos

- Python 3.6 o superior
- Linux / macOS / Windows (con WSL)
- Permisos de root (sudo)

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/Falconmx1/Mxcrack-ng.git
cd Mxcrack-ng

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar
sudo python3 mxcrack.py
