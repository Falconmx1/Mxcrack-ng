# 🔓 Mxcrack-ng

Herramienta de seguridad WiFi estilo aircrack-ng, pero sin necesidad de activar el modo monitor. Ideal para análisis rápidos y pruebas de penetración desde tu interfaz normal.

## 🚀 Características
- Captura de paquetes en redes cercanas
- Prueba de seguridad en redes WiFi
- Ataques básicos sin modo monitor
- Menú interactivo con 7 opciones

## 📋 Menú
1. Capturar paquetes  
2. Probar redes  
3. Ataques  
4. Herramientas  
5. Opciones  
6. Ayuda  
7. Salir  

## 🛠️ Requisitos
- Python 3.x
- Librerías: scapy, netifaces, colorama

## 📦 Instalación
```bash
git clone https://github.com/Falconmx1/Mxcrack-ng.git
cd Mxcrack-ng
pip install -r requirements.txt
python mxcrack.py
